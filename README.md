# Personal Portfolio Website

A modern, responsive personal portfolio website built with React and Material-UI.

## Features

- Responsive design
- Dark theme
- Smooth scrolling
- Animated sections
- Modern UI components
- Mobile-friendly navigation

## Technologies Used

- React.js
- Material-UI
- Framer Motion
- React Scroll
- Emotion (CSS-in-JS)

## Getting Started

1. Clone the repository
```bash
git clone <repository-url>
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```

## Customization

1. Update your personal information in the components
2. Modify the theme colors in `App.js`
3. Add or remove sections as needed
4. Update social media links in the Contact component
